# Task-management-system

###### Setup: Clone repository

###### Client App
**Step: 1.** go into client folder
**Step: 2.** Do npm i
**Step: 3.** npm start

###### Server App
**Step: 1.** Go into server folder
**Step: 2.** Do npm i
**Step: 3.** Add Env
`
    NODE_ENV=development
    PORT=9000
    CONNECTION_STRING=
    JWT_SECRET=
`
**Step: 4.** Do npm start
